# Praveen Kumar K · Portfolio

Welcome to the official repository for my developer portfolio. This application is a modern, responsive showcase of my skills, projects, and work, built with high-fidelity visual design, smooth animations, and a secure SQLite backend database to manage contact form submissions.

## 🚀 Features

- **Dynamic, Interactive UI**: Crafted with state-of-the-art styling, dark modes, and rich motion transitions using Framer Motion.
- **Graphic Design & Development Gallery**: Displays coding projects alongside graphic design work.
- **SQLite Database Persistence**: Secure server integration utilizing Express and `better-sqlite3` to store and manage contact inquiries.
- **Built-in Contact Message Viewer**: Admin dashboard to review, clean, and manage contact submissions directly.

---

## 🛠️ Technology Stack

### Frontend
- **Framework**: React 19 + Vite 8
- **Styling**: Tailwind CSS + Custom Animations
- **Icons & Motion**: Lucide React, React Icons, Framer Motion
- **Routing**: React Router DOM

### Backend & Storage
- **Server**: Express (Node.js)
- **Database**: SQLite (managed with `better-sqlite3`)
- **API**: Modern REST API endpoints for contact forms (`POST`, `GET`, `DELETE` operations)

---

## 💻 Local Setup & Development

Prerequisites:
- [Node.js](https://nodejs.org/) installed.

### 1. Install Dependencies
```bash
npm install
```

### 2. Run Backend Server
Start the Express server on [http://localhost:3001](http://localhost:3001):
```bash
npm run server
```

### 3. Run Frontend Development Server
Start the Vite developer environment on [http://localhost:5173](http://localhost:5173):
```bash
npm run dev
```

### 4. Admin Message Dashboard
Open [http://localhost:3001/](http://localhost:3001/) in your browser to access the interactive message logs and database management tool.
